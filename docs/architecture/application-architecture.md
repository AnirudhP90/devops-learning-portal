# DevOps Learning Portal Architecture

## Purpose

The DevOps Learning Portal is a centralized platform for storing and organizing:

* Roadmaps
* Study Materials
* Interview Questions
* Notes
* Projects

## Technology Stack

### Infrastructure

* AWS EC2
* Ubuntu Linux

### Backend

* Python
* Flask

### Frontend

* HTML
* CSS
* Bootstrap

### Version Control

* Git
* GitHub

## Current Architecture

```text
Browser
   │
   ▼
Flask Application
   │
   ▼
Templates
   │
   ▼
HTML Response
```
