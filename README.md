# DevOps Learning Portal

A personal learning portal built using Python Flask and AWS.

## Purpose

This portal is designed to centralize:

- DevOps Roadmaps
- Study Materials
- Interview Questions
- Personal Notes
- Projects
- Certification Preparation

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Bootstrap
- Git
- GitHub
- AWS EC2

## Project Structure

```text
devops-learning-portal
├── app.py
├── data
├── docs
├── static
├── templates
└── requirements.txt
```

## Setup

```bash
git clone <repo-url>
cd devops-learning-portal

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python app.py
```
